Top Level Design

The final product will be in a jar file. Users of this api just need to import basicj.*, or import basicj.BasicJ, and write a class that extends BasicJ, and finally run the class using the main method.

An example skeleton code is:

import basicj.*;

public class UserProgram extends BasicJ {
	public UserProgram() P\ {
		//BasicJ commands here
	}

	public static void main(String args[]) {
		new UserProgram();
	}
}
