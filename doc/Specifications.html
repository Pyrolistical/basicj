BasicJ Specifications

Functional Specifications

1. The "basic form" includes the following commands:

  1. print - prints without new line
  2. println - prints with new line
  3. input - prompts user for input and returns it
  4. color - sets the foreground color for all print/draw commands, and the background optionally. uses rgb or the standard 16 colors:
    1. black #000000
    2. (dark) blue #000080
    3. (dark) green #008000
    4. (dark) cyan #008080
    5. (dark) red #800000
    6. (dark) magenta #800080
    7. brown/(dark) yellow #808000
    8. [light] gray #c0c0c0
    9. (dark) gray #808080
    10. [light] blue #0000ff
    11. [light] green #00ff00
    12. [light] cyan #00ffff
    13. [light] red #ff0000
    14. [light] magenta #ff00ff
    15. yellow #ffff00
    16. white #ffffff
  5. point - draws a pixel at x, y
  6. line - draws a line from x0, y0 to x1, y1
  7. circle - draws a circle with center x, y and radius of r
  8. text - draws a string at x, y
  9. width - returns the width of the screen
  10. height - returns the height of the screen
  11. screen - sets the window size (the space inside the frame, not the size of the frame). also allows user to tell BasicJ to auto size (0,0)
  12. zoom - sets the zoom factor of the screen. for example zoom 2 draw represents 1 pixel by a 2x2 box of the same color
  13. abs - returns the absolution value of a float
  14. sqrt - returns the sqrt of a float
  15. floor - returns the (int) floor of a float
  16. ceil - returns the (int) ceil of a float
  17. round - rounds a float to the d decimal places
  18. max - returns the max of two floats
  19. min - returns the min of two floats
  20. sin - returns the sin of degrees
  21. cos - returns the cos of degrees
  22. tan - returns the tan of degrees
  23. asin - returns the arcsin in degrees
  24. acos - returns the arctan in degrees
  25. atan - returns the arctan in degrees
  26. atan2 - returns the arctan with parameters a, b in degrees; special case when b = 0
  27. pow - returns the power of base b to exponent e
  28. log - returns log base e
  29. time - returns the 24 hour time "HH:MM:SS"
  30. date - returns the date "YYYY-MM-DD"
  31. wait - pauses the execution of the program, in milliseconds
  32. rand - returns a random integer in a range or returns a float from 0..1
Note the screen origin at top-left corner of the window.

2. The commands print/draw to a Window, where the "active" space inside the window is the size set by the screen command

3. If the screen size (due to zoom or otherwise size), then the window should have scrollbars to pan the "screen"

4. Allow the Window to override the zoom, and reset it to the one set by the program

5. Since users are new to programming, errors messages must be clear and easy to understand

6. Favor warnings rather than compile exceptions

7. Intermediate level commands are:

  1. keypressed - returns the last key pressed, can also test for no keys pressed
  2. mouseclicked - returns the x,y of the last place clicked on the screen
  3. open - opens a file for writing
  4. write - writes to a file
  5. writeln - writes to a file with a new line
  6. close - closes the open file

Non-Functional Specifications

1. The final product is to be packaged into a Jar file

2. Documentation available to end-user should be in both JavaDoc and nicely formated manual (ie. the manual should group commands logically, where javadoc does not)
