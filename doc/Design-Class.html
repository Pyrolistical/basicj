Class Design

The BasicJ class extends JFrame, and holds all the logic for the menu and zoom override.

BasicJ delegates all the core api commands, except the wait command, to the follow classes:

  • Screen - this JComponent handles all the print/drawing, input and event commands (commands: 1a-l, 7a-b)
  • Math - this class handles the number related commands (commands: 1m-ag, except 1ae) and SPEC#8
  • FileIO - this class handles the file io commands (commands: 7c-f)
